# Clarence Linux Doc

Java 后端转嵌入式 Linux / 硬件方向的学习知识库，基于 [VuePress](https://v2.vuepress.vuejs.org/zh/) + [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/) 构建。

姊妹项目：[clarence-java-doc](../clarence-java-doc)（Java 后端知识体系）。

## 快速开始

```bash
npm install
npm run docs:dev     # 本地预览 http://localhost:1001/clarence-linux-doc/
npm run docs:build   # 构建产物在 docs/.vuepress/dist
```

## 学习路线

```
第一阶段（地基）：  Linux 基础 → Shell → 系统管理
第二阶段（核心）：  C 语言 → 系统编程 → 网络编程
第三阶段（嵌入式）：嵌入式 Linux → 硬件外设
第四阶段（进阶）：  驱动开发 → 内核机制
第五阶段（整合）：  工业协议 → IoT 网关 · 调试工具贯穿全程
```

目录结构与文档约定见 [CLAUDE.md](./CLAUDE.md)。
